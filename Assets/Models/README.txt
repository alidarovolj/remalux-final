# Модель сегментации стен

Для работы приложения необходима модель сегментации стен в формате ONNX.

## Рекомендуемые модели

1. **DeepLabV3+** с backbone MobileNetV2, обученная на датасете ADE20K или кастомном датасете стен
2. **SegFormer** - более современная и легкая архитектура
3. **U-Net** с легким энкодером

## Требования к модели

- Формат: ONNX
- Входное изображение: RGB, размером (1, 3, 513, 513) или (1, 3, 256, 256)
- Выход: маска сегментации с каналами для различных классов (включая стены)

## Подготовка модели

1. Обучите модель в PyTorch или TensorFlow
2. Экспортируйте модель в формат ONNX
3. Оптимизируйте модель с помощью инструментов ONNX (например, ONNX simplifier)
4. Поместите файл .onnx в эту папку
5. Укажите имя файла модели в компоненте WallSegmentation

## Тестирование

- Проверьте маску сегментации на тестовых изображениях
- Убедитесь, что модель корректно определяет стены в различных условиях освещения

## Дополнительно

Для улучшения производительности можно использовать:
- Квантование модели до INT8/FP16
- Замену тяжелых операторов на более легкие версии
- Пруннинг сети для уменьшения размера модели
